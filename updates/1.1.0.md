# What has been updated?

## Changelog - (Added, Fixed, Removed):

1. Fixed an issue with water barrel capacity, it could go above 100% capacity.
2. Added a new animal (pig) which does not have any actions, this is useful only for server owners who want to use different type of animals for different meat types. **NOTE** the pig does not have any feeding system at the moment, it only consumes water, in the next update i am going to implement a feeding system. If you encounter any errors or issues related to pigs, open a ticket. Also, **note** that there is no pig icon at the moment, it will also be released in the next update.
3. Added an option for goats and cows related to milking, you can now disable or enable the male cows or goats to be milked or not.
4. Added a new option for ranches to allow the owners to invite / add as members on a ranch only players who have the required job(s) - (Each ranch can have its own job requirements).
5. Added a new option for ranches to add a limit on the milk jug containers to hold a limited amount of liters than infinite (you can also have no limitation, its up to you).
6. Added item support for hay and corn to be replaced in case someone is not interested or does not want to use the hay or corn deliveries and wants the players to lose items instead for feeding the animals.
7. Replaced `AddHungerCapacityOnHaybaleDeliver` option which was on goats and cows configuration which should not because the hay capacity is supported for both animals and placed into `config_animals.lua` on the default configuration options section outside of goats and cows.
8. Fixed an issue on Corn Pickup Deliveries, in case the corn pickups had objects function system available, it would not be functional, the player no matter what, would not be able to see the configured objects.
9. Fixed a major issue related to dead and alive animals, the dead animals were counted and that caused many issues on spawning and throwing error on unknown position to be spawned which is normal because dead animals shouldnt be counted, they were taking a position that should be only for alive animals.
10. Fixed an issue on member permissions, the permissions would not be displayed properly on the selected member, it would display only the first member result than the target member.
11. Added some extra options related to HayFoodCoords for every ranch for displaying an icon or not and the display distance.

## How to update?

- You can easily update the modified files by redownloading from the **Cfx.re Portal** and copy-paste on the mentioned directory file paths.
- If new options have been implemented to the configuration file or edited, **DO NOT** copy-paste the new configuration file, otherwise you will get the latest configuration but you will lose all the changes you have made. The proper way updating a configuration file is by finding the new configuration options or the modified ones and do it **manually**.

*Cfx.re Portal **DOES NOT** force-update your configuration file or any files, if it did, you would always lose all the configuration changes you made or would totally break the script, all files are updated or edited manually.*
